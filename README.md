# Influencer Gear Hub

**Influencer Gear Hub** is a professional and visually appealing landing page designed for content creators to discover and purchase essential gears like cameras, microphones, and lighting setups.

## 🚀 Project Overview
This project was developed as part of my web development learning journey. The goal was to create a modern e-commerce-style landing page focusing on clean UI and responsive design principles.

## ✨ Features
- **Modern Hero Section:** Eye-catching design to grab attention.
- **Product Showcase:** Clean grid layout for displaying gears.
- **Responsive Design:** Optimized for mobile, tablet, and desktop screens.
- **Hover Effects:** Interactive UI elements for a better user experience.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling, Flexbox, and CSS Grid.
- **Google Fonts:** For modern typography.
